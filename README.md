# ATLAS Diagnostic

Site vitrine pour ATLAS — diagnostic stratégique pour TPE et PME.

## Déploiement sur Vercel

1. Créer un compte sur vercel.com
2. Connecter votre compte GitHub
3. Pusher ce projet sur un repo GitHub
4. Importer le repo dans Vercel
5. Vercel détecte automatiquement Next.js et déploie

## Développement local

```bash
npm install
npm run dev
```

Le site sera accessible sur http://localhost:3000
